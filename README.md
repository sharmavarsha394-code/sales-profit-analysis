# Sales and Profit Analysis

## Objective
To analyze product performance, monthly trends, and city-wise profitability.

## Tools Used
- Python
- Pandas
- Matplotlib

## Key Insights
- Smartwatch is the highest selling and most profitable product.
- March recorded the highest sales.
- Kolkata generated the highest sales.
- Chennai is the most profitable city.
